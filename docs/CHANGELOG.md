# 変更履歴

このプロジェクトのすべての重要な変更は、このファイルに記録されます。

フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に基づいており、
バージョン番号は [Semantic Versioning](https://semver.org/lang/ja/) に従っています。

## [Unreleased]

## [1.1.0] - 2026-05-02

### 追加
- トレイアイコン機能 - ウィンドウ最小化時にシステムトレイに最小化
- キーバインド処理をリファクタリング、add_hotkeyを使用した新しい実装
- 設定ファイルが見つからない場合の改善されたエラーメッセージ

### 変更
- keyboardモジュールをpynputに置き換え - キーボード入力の安定性を向上
- 置換辞書バックアップ設定を削除 - 設定の簡素化

### 削除
- **クリップボード出力機能を削除** - アプリケーションはGoogle Docsへの直接入力のみをサポートするように変更
- クリップボード関連の設定、UI要素、テストを削除

## [1.0.1] - 2026-04-30

### 追加
- Google Docsプレースホルダ機能 - 録音開始/停止時にプレースホルダを表示・置換・削除
- JSONファイル処理スクリプトにGUIファイル選択機能を追加

### 変更
- Google Docs出力モードのプレースホルダ待機時間を整数型に変更
- 自動停止時の通知メッセージと quit 処理を改善
- JSONファイル処理スクリプトの機能を改善
- UIコンポーネント（OutputMode Combobox）の初期化処理を改善

## [1.0.0] - 2026-04-29
- VoiceDocs の初版リリース
